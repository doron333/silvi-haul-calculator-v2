# Silvi Haul Calculator v2

Modern React + TypeScript haul rate calculator for aggregate trucking (NJ/PA/DE).

## Features
- Valhalla truck routing
- Live EIA diesel prices
- AI pricing insights (Claude)
- Responsive Tailwind UI
- Leaflet maps

## Setup
1. `npm install`
2. Copy `.env.example` to `.env` and add keys
3. `npm run dev`

Deployable on Vercel.
